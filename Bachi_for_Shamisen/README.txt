# Summary

I designed and printed this awesome bachi for my diy shamisen. It works great and prints really easy. 

I've printed in both ABS and Taulman alloy 910 and both work really great. 

# Print Settings

Printer: Sonhokey Prusa i3 2015
Rafts: No
Supports: No
Resolution: 0.2 layer height
Infill: 15%

Notes: 
These are the settings I used, however, any settings can be tweaked for you liking.
